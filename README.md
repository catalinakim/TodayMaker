# TodayMaker
SpringBoot, JPA를 활용한 사이드 프로젝트

### 기능
- 할일을 카테고리별로 관리합니다.
- 오늘 날짜에 할일들을 추가합니다.
- 오늘 할일 순서를 Drag & Drop으로 바꿔줍니다.
- 오늘 할일 우선순위 1,2,3위를 지정합니다.
- 오늘 할일 중요한 것은 별표⭐ 표시합니다.
